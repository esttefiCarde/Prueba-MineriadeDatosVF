# Análisis de Salarios

Este proyecto corresponde a un análisis exploratorio de datos (EDA) sobre factores que influyen en el salario.

## Contenido
- Limpieza de datos
- Análisis exploratorio
- Correlaciones
- Transformación de variables

## Herramientas utilizadas
- Python
- Pandas
- Seaborn
- Scikit-learn
